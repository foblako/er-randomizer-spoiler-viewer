# Elden Ring Randomizer — Spoiler Log Viewer

A lightweight desktop GUI for browsing spoiler logs generated by the [Elden Ring Item and Enemy Randomizer](https://www.nexusmods.com/eldenring/mods/428).

![screenshot placeholder](https://via.placeholder.com/800x480?text=Screenshot+coming+soon)

## Features

- Opens any randomizer spoiler `.txt` log
- Built-in filters: Great Runes, Remembrances, Key Items, Boss Placements, and more
- Full-text search across entries
- Shopping-cart selection: tick items and save as a reusable filter
- User-saved filters persist across sessions and work on any log with the same options
- Light / dark theme toggle
- Copy results to clipboard or save to file
- File association friendly — pass the log path as a CLI argument

## Requirements

- Python 3.11+
- Tkinter (included with the standard Python installer on Windows and macOS)

No third-party packages required.

## Running from source

```bash
git clone https://github.com/foblako/er-randomizer-spoiler-viewer.git
cd er-randomizer-spoiler-viewer

python -m er_randomizer
# or
python er_randomizer_viewer.py
# or pass a log directly
python -m er_randomizer path/to/spoiler_log.txt
```

## Installing as a package

```bash
pip install -e .
er-spoiler-viewer
```

## Project layout

```
er_randomizer/
  __init__.py        — public API
  __main__.py        — entry point for python -m er_randomizer
  gui.py             — Tkinter application
  parser.py          — spoiler log parser
  filters.py         — built-in filter definitions
  formatting.py      — entry → tagged text chunks
  presets.py         — category labels
  theme.py           — light/dark theme palettes
  config.py          — ~/.er_randomizer.json store
  user_presets.py    — user-saved filter persistence
er_randomizer_viewer.py  — legacy launcher script
samples/             — example spoiler logs for testing
```
